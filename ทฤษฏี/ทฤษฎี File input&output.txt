File input/output
    * เพื่อให้โปรแกรมเก็บข้อมูลระหว่างเวลาที่รัน เราควรจะเซฟข้อมูล
        - ข้อมูลจะถูกบันทึกลงในไฟล์ โดยทั่วไปจะอยู่ในดิสก์คอมพิวเตอร์
        - ข้อมูลที่ถูกเก็บสามารถดึงข้อมูลมาใช้ได้
    * การเขียนข้อมูล: เพื่อบันทึกข้อมูลลงไฟล์
    * การแสดงผลไฟล์: ไฟล์ที่ข้อมูลถูกเขียนไว้
    * อ่านข้อมูลจาก: การประมวลผลการดึงข้อมูลจากไฟล์
    * ไฟล์อินพุต: ไฟล์ที่ใช้อ่านข้อมูล

Type of Files
    โดยทั่วไป แบ่งเป็น 2 ประเภท:
        * text file: ประกอบด้วยข้อมูลที่เป็นข้อความ
        * binary file: ประกอบด้วยข้อมูลที่ยังไม่ถูกแปลงให้เป็นข้อความ
    
    วิธีในการเข้าถึงข้อมูลที่เก็บไว้ในไฟล์มี 2 วิธี
        * Sequential access: เป็นการอ่านไฟล์ตามลำดับตั้งแต่ต้นจนจบ ไม่สามารถอ่านข้ามได้
        * Direct access: สามารถข้ามไปอ่านข้อมูลที่ต้องการในไฟล์ได้

With Statement
    with statement  ใช้เพื่อให้แน่ใจว่าไฟล์จะถูกปิดอย่างถูกต้องหลังจากที่โปรแกรมทำงานเสร็จสมบูรณ์ 
                    แม้ว่าจะเกิดข้อยกเว้น (exception) ขึ้นก็ตาม การใช้คำสั่ง with ช่วยให้จัดการทรัพยากรไฟล์ได้อย่างมีประสิทธิภาพมากขึ้น 
                    เพราะจะปิดไฟล์โดยอัตโนมัติเมื่อโค้ดในบล็อกนั้นถูกออกจากการทำงาน ซึ่งช่วยลดความจำเป็นในการเรียกใช้คำสั่ง close() โดยตรง 
                    และลดความเสี่ยงในการเปิดไฟล์ทิ้งไว้ ซึ่งส่งผลให้โค้ดดูสะอาดและอ่านง่ายขึ้น
